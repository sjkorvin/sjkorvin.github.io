Privacy

This site uses Google Analytics for enhanced tracking. This site also makes use of Google fonts.

This site is hosted on GitHub and comments are provided by Utterances, which creates comments through GitHub issues. Thus, this site assumes GitHub’s privacy policy.

This site’s domain, stevenkorvin.com, is registered on Gandi. Please see Gandi’s privacy policy if you have any questions about Gandi’s handling of user privacy and data.

I have provided a link to my personal Linktree in the description within this site’s homepage. Usage of Linktree is subject to Linktree’s privacy policy. I make use of Google Analytics on my Linktree profile.

I may use shortcodes built in to Hugo, which this site is built with, on site posts to embed content. I have already done this with my public key in a GitHub gist. If I use these shortcodes to embed content, you are subject to the privacy policies of the websites with which I am embedding content.

Sometimes, I shorten links using kutt.it. These links will begin with link.stevenkorvin.com. I get some analytics when those links are visited. Please see kutt’s privacy policy for more information.

My hashnode blog at hashnode.stevenkorvin.com, is subject to Hashnode’s privacy policy. I make use of Google Analytics and Facebook Pixel ID with my Hashnode blog.

The contact form on this website is made possible by Formspree. Please see Formspree’s privacy policy is you have any questions on how Formspree processes user data. My Formspree form makes use of Google ReCAPTCHA.

This site is being served through Cloudflare nameservers and many options in Cloudflare’s free plan have been applied to this site.
